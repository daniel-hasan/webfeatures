Django
O que é? Framework gratuito e de código aberto para a criação de aplicações web, escrito em Python.
django-admin É um script que cria a estrutura básica do diretório Django na qual não se deve apagar.
Diretório/Arquivos básicos manage.py é um script que facilita a gestão do site. Ele coloca o pacote do projeto no sys.path e define a variável ambiente DJANGO_SETTINGS_MODULE que aponta para o arquivo settings.py
/settings.py é um script que contém as configurações do projeto Django
/urls.py é um arquivo que contém uma lista de padrões usados por urlresolver (que funciona como um carteiro que verifica o endereço da correspondencia para enviá-la)
Administração Django É uma interface intuitiva de administração automática que o Django oferece que permite trabalhar com permissões de acesso e customização de telas. O Admin cria um log, registrando todas as alterações realizadas.
Permite obter uma interface ainda mais agradável, eficiente e com uma navegação melhorada. Para isso, basta utilizar módulos externos, como o django-grappelli.
Para fazer login é necessário um superuser - um usuário que possui controle sobre tudo do site. Assim, quando acessado http://127.0.0.1:8000/admin/ é possível aproveitar de todos os recursos oferecidos pelo Django.
Modelos Django Um modelo no Django é um tipo especial de objeto - ele é salvo em um banco de dados. Um banco de dados é uma coleção de dados.
Templates É um arquivo que se pode reutilizar para apresentar diferentes informações de uma forma consistente. O formato do template do Django é descrito na linguagem HTML.
Views Local onde se escontra a "lógica" da aplicação. Ele irá solicitar informações a partir do model e passá-lo para um template. Views, no fundo, não passam de métodos escritos em Python.
QuerySet É uma lista de objetos de um dado modelo que permite a leitura dos dados do banco, filtre e ordene o mesmo.
Algumas tags de template Django:
Variáveis { { variavel } }

Exibe variável no Django template
Loops { % for .. in .. % } ... { % endfor % }

Loops no Django template.
Blocos { % block content % } ... { % endblock % }

É uma tag de template que permite a inserção de html neste bloco em outros templates que estendem um base.
Condicional { % if ... % } ... { % endif % }

tags de condição como if , else ...
Herança { % extends 'caminho/index.html' % }

Estende um template em outro arquivo.
Arquivos Estáticos { % load staticfiles % }

Carrega arquivos estáticos.
São todas as imagens e arquivos CSS -- arquivos que não são dinâmicos, então seu conteúdo não depende do contexto da requisição e será o mesmo para todos os usuários.
Um exemplo de facilidade do Django
Formulários Com Django se pode tanto criar um formulário do zero quanto criar um ModelForm que salva o resultado do formulário para um determinado modelo. O framework oferece o módulo de formulários : from django import forms.
